# Working Dossier Files

This folder holds evolving promotion-case construction files. These are not final dossier prose and not the public-facing repository landing page.

Current working files:

- `master-dossier-skeleton.md` builds the first promotion-case architecture from the ASU Vita export, public-facing README, short CVs, and SciENcv sketches.
- `vita-section-map.md` maps the ASU Vita export into usable dossier evidence categories.

The working rule is to build evidence architecture first, then turn that evidence into affirmative promotion prose later.
