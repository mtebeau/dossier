## Post-Hoc Review Process

Post-hoc (or post-project) review is recommended in the SHPRS/History promotion criteria for scholarship that did not undergo traditional peer review before release. It is explicitly distinct from formal external tenure-review letters.

### Purpose

A post-hoc reviewer assesses a specific project's scholarly significance, quality, methodological contribution, role in the field, and public or professional impact. The letter speaks to the project as scholarship — not to the candidate's overall record.

### How It Differs from External Reviewer Letters

| Post-hoc reviewer | External tenure reviewer |
| --- | --- |
| Evaluates one project | Evaluates overall scholarly record |
| Speaks to project as scholarship | Speaks to national/international standing |
| Letter attached to dossier as evidence | Letter submitted through formal review process |
| Can be someone with a professional relationship to the project | Must meet independence requirements |
| Can be solicited before formal review cycle | Solicited in spring 2027 |

### Packet Contents (per project)

Each request packet should include:

- Cover letter explaining purpose (dossier evidence; not formal tenure review letter)
- Project description (origin, scope, Mark's intellectual and organizational role)
- Selected evidence (scale metrics, press/uptake, partner/adoption evidence, selected publications)
- Evaluation questions (see below)
- Deadline and format guidance (suggested: 2–4 pages; deadline 8 weeks from request)

### Evaluation Questions (draft)

1. What is the scholarly significance of this project in its field?
2. What did the project contribute to historical practice, methods, or public/community knowledge?
3. How should the project be evaluated as scholarship, given that it took a digital/public/archival rather than conventional printed form?
4. What evidence of quality, rigor, and impact do you observe?
5. What was the candidate's intellectual role in the project's conception, design, and execution?
6. Are there comparable projects in the field, and how does this project compare?

### Reviewer Profile

- Post-hoc reviewers can have professional familiarity with the project (unlike tenure reviewers)
- They should be senior scholars, archivists, or practitioners with standing in the relevant field
- Check: no current conflicts of interest; no co-authorship or supervisory relationship with Mark
- Record each reviewer's affiliation, fit, and relationship note in the tracker
